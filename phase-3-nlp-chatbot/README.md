# Task Management Chatbot (Phase 3)

A natural language chatbot that can understand and manage your tasks through conversational commands.

## Features
- Add tasks using natural language
- Delete tasks with simple commands
- Mark tasks as complete/incomplete
- List all tasks
- Understand various command formats

## Technology Stack
- TypeScript/JavaScript
- Natural Language Processing
- Simple local storage (no external APIs required)
- React/Vue frontend or simple HTML/CSS/JS

## Commands Supported
- "Add task: Buy groceries" or "Create task: Finish report"
- "Delete task: Buy groceries" or "Remove task: Finish report"
- "Complete task: Buy groceries" or "Mark task: Buy groceries as done"
- "List tasks" or "Show all tasks"
- "What's on my list?" or "Show me my tasks"
